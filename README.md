# PRODIGY_DS_01

## Task 1: Population Distribution Visualization

### Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Dataset
- World Bank Population Dataset
- Indicator: Total Population

### Technologies Used
- Python
- Pandas
- Matplotlib

### Project Description
This project analyzes the World Bank Population dataset and visualizes the **Top 10 Most Populated Countries (2023)** using a bar chart. The dataset was cleaned to remove regional and aggregate entries so that only individual countries are displayed.

### Output
The generated bar chart is saved as:

`population_distribution.png`

### How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the Python script:

```bash
py task1.py
```
### Sample Output

A bar chart displaying the **Top 10 Most Populated Countries (2023)**.
